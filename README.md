## Чек-лист реализованных проверок для сайта https://www.saucedemo.com/ :

### 1. Авторизация (Login)
- [ ] Успешная авторизация под валидным пользователем (переход на страницу Products)
- [ ] Ошибка авторизации при пустом поле "Username"
- [ ] Ошибка авторизации при пустом поле "Password"
- [ ] Ошибка авторизации при вводе невалидных или заблокированных учетных данных

### 2. Корзина (Cart)
- [ ] Добавление товара в корзину: проверка отображения корректного названия и цены
- [ ] Удаление добавленного товара из корзины.

### 3. Оформление заказа: Информация (Checkout: Your Information)
- [ ] Успешный ввод валидных данных (First Name, Last Name, Postal Code) и переход на следующий шаг
- [ ] Появление ошибки при пустом поле "First Name"
- [ ] Появление ошибки при пустом поле "Last Name"
- [ ] Появление ошибки при пустом поле "Postal Code"
- [ ] Корректная работа кнопки "Cancel" (возврат на страницу корзины)

### 4. Подтверждение заказа (Checkout: Overview)
- [ ] Проверка соответствия названия товара и его цены на финальном шаге проверки заказа
- [ ] Успешное завершение заказа: клик по кнопке Finish и отображение финального заголовка "Checkout: Complete!"
